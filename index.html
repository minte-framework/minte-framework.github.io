About

MINTE is a semantic data integration framework that collects and integrates heterogeneous data sources into an actionable knowledge graph (i.e., organizations can build valuable services on top of the knowledge graph such as Semantic Search, Analytics, and Conversational Assistants). MINTE relies on both: knowledge stated in RDF vocabularies and semantic similarity measures to merge heterogeneous data into a consolidated RDF graph. MINTE follows a two-fold approach to solve the problem of integrating RDF graphs. In the first step, MINTE transforms data into RDF entities, then MINTE implements a 1-1 weighted matching algorithm to identify semantically equivalent RDF entities in different data sources. Then, MINTE relies on different fusion policies to integrate triples from these semantically equivalent RDF entities.

Publications

  • Diego Collarana, Mikhail Galkin, Christoph Lange, Simon Scerri, Sören Auer, Maria-Esther Vidal: Synthesizing Knowledge Graphs from Web Sources with the MINTE+ Framework. International Semantic Web Conference (2) 2018: 359-375
